Futebol paralímpico

Regras do paralimpico

Link

futebol paralímpico consiste na adaptação do futebol regular para atletas com alguma deficiência. Estes desportos são jogados, normalmente, usando regras da Fifa, com modificações quanto ao terreno de jogo, equipamento, número de jogadores, e outras regras necessárias para tornar o jogo adequado aos atletas.

Quais são os tipos de Futebol paralímpico?

duas principais versões do futebol paralímpico são o futebol de cinco, para atletas com deficiências visuais, e o futebol de sete, para atletas com paralisia cerebral.

Qual é a importância do esporte paralímpico?

A atividade esportiva contribui não só para o desenvolvimento físico, como também é uma poderosa ferramenta de ajuda na reabilitação e inclusão social de pessoas com deficiência.
